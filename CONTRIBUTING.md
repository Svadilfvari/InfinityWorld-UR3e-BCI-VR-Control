# Contributing

This repo is a research prototype. Contributions are welcome, especially:

- Improving IK robustness (multiple solutions, singularity handling)
- Replacing Euler-based orientation with quaternions / rotation matrices
- Making the TCP protocol framed (newline or length-prefixed)
- Adding a real UR3e interface (RTDE / URScript) while keeping Unity simulation
- Cleaning up EEG inference (proper scaler, montage mapping, online decoding)
